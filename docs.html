The Art API

Geting an art from API

To Get a random art from the API, you need to send a GET request to the following URL:
https:///api

Uploading an art to the API

To upload an art to the API, you need to send a POST request to the following URL:
https:///api
If you get a 400 Bad Request look at the response body for more information,

After you have uploaded an art make sure you store the ID and Pass for later.

Deleting your art

To delete your art, you need to send a DELETE request to the following URL:
https:///api/YOUR_ART_ID
Replace YOUR_ART_ID with the ID you got from the API after uploading, If you get a 401 Unauthorized response then make sure to send this in the request body:
{"pass": "YOUR_PASSWORD"}
Replace YOUR_PASSWORD with the Pass you picked while uploading, If you cant delete it then send us an email at art@mail.sx9.is-a.dev
